# levi-vue-admin
## 简介
本项目主要基于vue3.0 + vite + antd-design-vue

## 快速开始
```
yarn // 安装
yarn dev  // 本地运行
yarn build // 打包
```
## 目录结构
```
├── README.md
├── createMenu.js
├── index.html
├── package.json
├── public // 一些全局的文件
│   └── favicon.ico
├── src // 主要代码文件夹
│   ├── App.tsx
│   ├── apis // 存放和后端连接的文件
│   ├── assets // 存放图片，字体等文件
│   ├── components // vue组件存放目录
│   ├── config.ts // 配置
│   ├── hooks
│   ├── layouts // 页面布局文件夹
│   ├── main.ts 
│   ├── pages // 页面文件夹
│   ├── permission.ts
│   ├── router
│   ├── store // vuex操纵数据
│   ├── styles
│   ├── types
│   ├── useApp.ts
│   ├── utils // 辅助文件夹
│   └── views // 主要页面逻辑
├── tsconfig.json
├── vite.config.ts
├── yarn-error.log
└── yarn.lock
```
